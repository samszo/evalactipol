Cette classe permet de construire, avec php, le code Xul pour les trees et les listbox

Cette classe permet de construire à partir des variables Ajax le contenu de l'interface Xul. Elle calcule la valeur des variables qui s'affichent dans l'interface de gestion de l'application. Ceci se fait en consultatnt la base de données pour extraire les informations, les construire selon le format Xul et les faire retourner sous format Xul pour l'affichage par la suite.
